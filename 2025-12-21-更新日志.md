## 📅 2025-12-21 重大更新：域名迁移

### 🔄 基础设施 (Infrastructure)
- **域名变更**: 正式从 `yangzhen.de5.net` 迁移至新顶级域名 `yangzhenai.top`。
- **Cloudflare 深度配置**: 
  - **SSL**: 开启 Full (Strict) 模式，强制 Always Use HTTPS，确保全站银行级加密。
  - **Redirects**: 配置 Page Rules 实现 `www.yangzhenai.top` 301 永久重定向至主域名。
  - **Protocol**: 启用 HTTP/3 (QUIC) 和 0-RTT 连接复用，大幅降低延迟。
  - **Compression**: 开启 Brotli 智能压缩。
  - **Monitoring**: 开启 Real User Monitoring (RUM) 真实用户性能监控。

### 🔎 SEO 专项优化 (SEO)
- **所有权验证**:仅仅耗时30分钟，完成了 Google、Bing、百度三大搜索引擎的站点验证。

- **结构化数据**: 修复全站 JSON-LD schema (WebSite, Person, Article) 中的旧域名引用。
- **百度收录**: 在 `PublicLayout` 中集成了百度最新的 HTML 验证标签 `codeva-PGmpZDM43W`。

## 📅 2025-12-16 更新回顾：前端与性能

### ⚡ 性能优化 (Performance)
- **资源加载**: 实现首屏关键资源 (Avatar, CSS) 预加载 (Preload)。
